<!-- GETTING STARTED -->

## 快速开始

### 安装

#### 安装依赖

```
yarn install
```

#### 本地服务

```
yarn serve
```

#### 生产环境

```
yarn build
```

#### 单元测试

```
yarn test:unit
```
